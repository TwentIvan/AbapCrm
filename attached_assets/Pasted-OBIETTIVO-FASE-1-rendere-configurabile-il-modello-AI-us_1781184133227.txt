1. NUOVE TABELLE (shared/schema.ts)

ai_providers:
- id uuid PK default gen_random_uuid()
- name text not null (es. "OpenAI", "Anthropic", "Google", "DeepSeek")
- gateway_prefix text (es. "openai/", "anthropic/")
- enabled boolean default true

ai_models:
- id uuid PK
- provider_id uuid FK -> ai_providers
- model_key text not null unique (es. "openai/gpt-5", "anthropic/claude-sonnet-4-6")
- display_name text not null
- version text
- price_input_per_mtok decimal(10,4)  -- USD per milione di token input
- price_output_per_mtok decimal(10,4)
- context_window integer
- capabilities jsonb default '{}'  -- { "toolUse": true, "vision": true }
- status text default 'active'  -- active | deprecated
- synced_at timestamp

Crea uno script di seed (scripts/seed-ai-models.ts) che popola i 4 provider e almeno 8 modelli rappresentativi (2 per provider) con prezzi indicativi e capabilities. I prezzi sono placeholder aggiornabili.

2. GATEWAY CLIENT (server/ai-gateway.ts)

Crea un modulo unico per tutte le chiamate LLM con questa interfaccia:

  aiGateway.complete({
    modelKey: string,            // es. "openai/gpt-5"
    messages: ChatMessage[],
    responseFormat?: "json" | "text",
    tools?: ToolDefinition[],    // formato function calling OpenAI-compatible
    maxTokens?: number
  }): Promise<{
    content: string,
    toolCalls: ToolCall[],
    usage: { promptTokens, completionTokens, totalCostUsd }
  }>

Comportamento:
- Se è definita la env AI_GATEWAY_BASE_URL: usa il client OpenAI SDK puntato a quella base URL con AI_GATEWAY_API_KEY, passando modelKey come model (è un proxy LiteLLM, API OpenAI-compatible).
- ALTRIMENTI (fallback, deve funzionare da subito): se modelKey inizia con "openai/", chiama direttamente OpenAI con OPENAI_API_KEY rimuovendo il prefisso. Per altri prefissi senza gateway configurato, lancia un errore chiaro: "Gateway non configurato: il modello X richiede AI_GATEWAY_BASE_URL".
- Calcola totalCostUsd da usage e dai prezzi in ai_models (lookup per modelKey).
- Logga ogni chiamata: modelKey, token, costo, durata, servizio chiamante.
